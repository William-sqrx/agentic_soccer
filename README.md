# Agentic Soccer

Agentic Soccer consists of a Flask backend and a frontend client.

## Structure

```bash
agentic soccer/
├── graph/
│   ├── app.py
│   ├── graph.py
│   ├── main.py
│   ├── state.py
│   └── soccer-chat/
└── README.md
How to Run

Put the required environment variables in .env

Start the backend
cd graph
python3 main.py

Start the frontend
cd graph/soccer-chat
npm run start
